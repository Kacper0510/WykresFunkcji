O projekcie

Miejsca zerowe funkcji – argumenty, dla których funkcja przyjmuje wartość 0.

Funkcja liniowa

f(x)=ax+b

Miejsce zerowe funkcji liniowej, czyli przecięcie prostej f(x) i osi OX liczymy ze wzoru:

-b a

a - współczynnik kierunkowy
a ≠ 0 - jedno miejsce zerowe
a = 0 oraz b = 0 - nieskończenie wiele miejsc zerowych
a = 0 oraz b ≠ 0 - brak miejsc zerowych

b - wyraz wolny

Funkcja kwadratowa

f(x)=ax²+bx+c

Δ = b²-4ac

Δ<0 - brak pierwiastków rzeczywistych
Δ=0 - jeden pierwiastek rzeczywisty

x₁= -b 2a

Δ>0 - dwa pierwiastki rzeczywiste

x₁= -b-√Δ 2a

x₂= -b+√Δ 2a

a, b, c - współczynniki rzeczywiste, a ≠ 0

Funkcja wymierna

f(x)= W(x) G(x)

Miejsca zerowe funkcji wymiernej uzyskujemy obliczając pierwiastki wielomianu W(x)

W(x) - dowolny wielomian
G(x) - dowolny wielomian niezerowy

Funkcja wykładnicza

f(x)=ax

Funkcja wykładnicza posiada miejsca zerowe jedynie po przesunięciu o wektor.

a - liczba rzeczywista większa od 0

Funkcja logarytmiczna

f(x)=logax

Funkcja logarytmiczna ma zawsze jedno miejsce zerowe

x=1

a - liczba rzeczywista, a > 0, a ≠ 1
x > 0
